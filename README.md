# React + Vite Starter 🚀

一个简洁的 React + Vite 前端启动模板。

## 技术栈

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器（默认 http://localhost:3000）
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 项目结构

```
react-vite-starter/
├── index.html          # HTML 入口
├── vite.config.js      # Vite 配置
├── package.json
└── src/
    ├── main.jsx        # React 挂载入口
    ├── App.jsx         # 根组件
    ├── App.css         # 组件样式
    └── index.css       # 全局样式
```
